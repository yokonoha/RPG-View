画像では紹介していませんが、3人のキャラクター画像を右、左、中央に配置できます。  
There are 3 picture boxes to show 3 person images.(Right Center Left)
ダイアログは1文字ずつ話すように出現するようになっています。お好みでSE(サウンドエフェクト)をつける というのも良さそうです。  
The text will appear as if you were speaking one letter at a time. It may be a good idea to add SE (sound effects) to the dialog if you like.  

<img src="./1 (4).png" width="40%"></img>  
会話が終了すると、10秒間"偽"ローディング画面が表示されます。ここでは16:9の画像の表示が可能です。 ゲーム開始前のやり方紹介やルール説明を画像にしてグラフィカルにエンドユーザーに見せることができます。  
When the conversation ends, a “fake” loading screen is displayed for 10 seconds. Here, 16:9 images can be displayed. This allows the end-user to graphically see an introduction of how the game is played and an explanation of the rules before the game begins.
